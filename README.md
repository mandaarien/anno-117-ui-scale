![Gameplay UI 4K Example](/img/preview_1.jpg)
![Menu UI Example](/img/preview_3.jpg)
![Menu UI Example](/img/preview_2.jpg)

This mod changes  `<GUID="50286" Path="Values/UIConfigScaling/Scalings">` to:

```
MonitorSmall          0.88 --> 0.6875   (11/16)
MonitorMedium           1 -->  0.8125   (13/16)
MonitorBig            1.125 --> 1.000      (16/16)
TelevisionSmall       1.25 --> 1.125    (18/16)
TelevisionMedium 1.375 -->1.25     (20/16)
TelevisionBig             1.5 --> 1.375    (22/16)
```

for better use of the given screenspace on high resoultion displays, as I found the vanilla scalling wastefull big and annoying :)

You can change the preset values in the assets.xml file included in the mod.

 

## How To Use
Manual Install:
Download the mod via the [mod.io](https://mod.io/g/anno-117-pax-romana/m/smaller-ui-sclae#description) download function.
Unpack the zip file into one of the following Anno 117 mod directories:
– “C:/User/$Username$/Documents/Anno 117 – Pax Romana/mods”
– “Path/to/Game/Installation/Anno 117 – Pax Romana/mods” (use the ‘Open Game Directory’ button on your launcher to access with ease.)
